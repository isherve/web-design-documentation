HTML Basics

HTML stands for HyperText Markup Language.

It is the standard language for creating web pages.

HTML elements are represented by tags.

  • Common tags include:
    • <div>
    • <p>
    • <a>
  • HTML documents have a structure.
  • Each document starts with a <!DOCTYPE> declaration.
CSS Fundamentals

CSS stands for Cascading Style Sheets.

It is used to style and layout web pages.

CSS can control the color, font, layout, and even animations.

  • Selectors target HTML elements; for example:
    • h1 targets all <h1> elements.
    • .className targets elements with a specific class.
  • Styles are defined within rulesets.
JavaScript Overview

JavaScript is a programming language used to create dynamic and interactive content.

It can be embedded directly within HTML pages.

JavaScript can manipulate the DOM, allowing you to change HTML and CSS.

  • Common uses include:
    • Form validation
    • Creating dynamic content
  • An example of a simple command is console.log("Hello World!");.
Responsive Design

Responsive design ensures that web pages look good on all devices.

This involves using flexible layouts and media queries.

Media queries allow CSS to apply different styling based on the screen size.

  • Common breakpoints include:
    • 768px for tablets
    • 1024px for desktops
  • Design should adapt to the user’s device.
Accessibility

Web accessibility is about making websites usable for all people, including those with disabilities.

This includes using semantic HTML to improve screen reader compatibility.

It’s important to provide alternative text for images.

  • Consider color contrast and keyboard navigation.
  • Adhering to standards like WCAG increases inclusivity.
